Wishlist App – SE Project

A simple full-stack wishlist management system built as part of the Software Engineering course. The project allows users to create, update, and manage personal wishlists.

Overview

The system consists of:

Backend (PHP – Flight framework) for handling API endpoints

MySQL database for data storage

Frontend (HTML, CSS, JavaScript) for user interaction

Node modules for project tooling

Features

User login & registration

Create and manage wishlists

Add, edit, delete wishlist items

REST API built in PHP (Flight + PDO)

Simple and clean web interface

Tech Stack

Backend: PHP, FlightPHP, PDO, MySQL

Frontend: HTML, CSS, JavaScript

Tools: Postman, Composer, GitHub

Structure
backend/       → PHP API (routes, services, DAO)
frontend/      → HTML/CSS/JS UI
test/          → test files 
node_modules/  → project tooling dependencies
package.json   → metadata for project scripts

Running the Project

Import MySQL database.

Start backend using Apache or PHP server:

php -S localhost:8080 -t backend


Open frontend/index.html in browser.

Use Postman for testing API endpoints.
